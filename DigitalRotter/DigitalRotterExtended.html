Digital Rotter by LandonPET


This is inspired by the phenomenon of Digital rot
    which can be caused by cosmic rays flipping random bits in data overtime


      Supported formats for previewing in browser

      Browser supported images
      Browser supported videos
      Browser supported audios
      Text files
      .mid/midi files
      .nes rom files

      Note that even if a file isn't supported for previewing
      you can still put it in and download it.
      ANY file can be used for this purpose
      (but be careful!)
    

      JS libraries used
      Nostalgist for NES ROMs
      html-midi-player for MIDI
